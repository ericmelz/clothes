How this works

This site is data-driven. All clothing items live in data/items.json. Thumbnails and full images are referenced from the thumbs/ and images/ folders.

Quick start

  1. Place your source photos (HEIC/JPG/PNG) under a folder structure like source_photos/Shirts/IMG_1234.HEIC.
  2. Install the generator's dependencies (Python 3.11+): pip install pillow pillow-heif.
  3. Run python build/generate.py --src ./source_photos --out . from the site root. This will create WebP images and update data/items.json.
  4. Open index.html in your browser.

Editing metadata

You can edit titles and notes directly in data/items.json. As a lightweight option, you can also tweak notes per-item in the Details page; these changes are saved to your browser’s local storage. Use the “Export My Notes” button to download your changes as JSON and then merge them back into data/items.json if you want them to be permanent.

Why WebP?

Browsers vary in HEIC support. The generator converts your originals to WebP (good quality, small size) and creates thumbnails for fast grids.

Custom categories

Categories are inferred from the source folder names (e.g., Shirts, Pants). You can also add or rename categories by editing the JSON.
